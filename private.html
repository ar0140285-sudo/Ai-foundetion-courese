Highest Paying Job (bca)

📚 PHASE 1: BCA (3 Years)

🟢 SEMESTER 1 (Abhi - july)

College Subjects :

BCA Phase (2025–2028) Semester 1 abhi shuru hoga. Isme C/C++ aur Python pe focus karna. College ke subjects mein 90%+ attendance rakhna aur assignments time pe submit karna — yahi 9+ CGPA ka secret hai. Side mein daily 1 ghanta Apna College YouTube se Python seekhna. Koi project nahi banana abhi — sirf foundation strong karo. Semester 2 mein Data Structures aur DBMS aayega. Yeh do subjects bahut important hain — isko seriously lena. SQL bhi seekhna shuru karo kyunki yeh har company mein use hota hai. LeetCode pe daily 1 easy problem solve karna shuru karo. GitHub account bana lo is semester mein. Semester 3 mein HTML, CSS, JavaScript aayega college mein — yeh tere liye golden semester hai. Isme college ke saath saath 3 projects banana — ek portfolio website, ek to-do app, aur ek weather app jo real API use kare. Pehli baar kuch live deploy hoga internet pe — bahut achha feel hoga. Git/GitHub properly seekh lo is semester mein. Semester 4 mein Python aur Java aayega. Python toh already strong hoga, toh college mein easy rahega. Side mein backend development seekhna shuru karo — Flask ya Node.js. Is semester ke end mein resume bana lo aur Internshala pe internship apply karna shuru karo. Koi bhi remote internship mil jaaye toh zyada nahi sochna — kar lo. Semester 5 mein PHP aur Mobile Development aayega. PHP college ke liye padh lo, but side mein React.js seekho seriously. Yeh semester mein ek full stack project banana — frontend React, backend Node.js, database MySQL. Yeh project deploy karo cloud pe. Yeh tera sabse strong project hoga resume mein. Semester 6 mein Cloud Computing aayega — AWS basics isko seriously seekhna kyunki market mein demand bahut hai. Final project college ka outstanding banana — achhi documentation ke saath. Is semester mein placement drives mein actively participate karna. Target companies: TCS Digital, Cognizant GenC Next, ya koi startup jo ₹10L+ de. GATE Preparation (BCA ke Last 6 Months) Semester 5 ke end se GATE preparation shuru karna. Pehle 2 mahine un subjects ka revision karo jo BCA mein tha — DSA, OS, DBMS, Networks. Yeh sab already pata hoga toh revision easy hogi. Agle 2 mahine naye subjects seekhna — Theory of Computation, Compiler Design, Discrete Math. Last 2 mahine sirf previous year papers solve karna — 10 saal ke papers solve kar lo, pattern samajh aayega. Target simple hai — GATE mein 90+ percentile lao toh SGSITS Indore milega jo government college hai aur achha hai. 95+ percentile lao toh IIT Indore possible hai jo tera dream hona chahiye. MCA Phase (2028–2030) College priority Indore mein: Pehla choice IIT Indore hai — Simrol mein hai, Indore se 25 km. Package ₹18L–₹35L milta hai. GATE 95+ percentile chahiye. Doosra choice SGSITS Indore hai — government college, city mein hi hai, fees bahut kam hai, package ₹12L–₹18L. GATE 90+ percentile chahiye. Yeh most realistic target hai tera. Teesra choice SAIT Indore hai — GATE 75+ percentile chahiye, package ₹8L–₹12L. Backup option. MCA mein AI/ML specialization lo. Yeh field 2028–2030 mein aur bhi badi hogi. Python aur TensorFlow/PyTorch seekhna side mein. Final project AI/ML pe banana — yeh companies ko bahut attract karta hai. Job Phase (2030 Ke Baad) MCA ke baad directly top companies apply karna. Realistic targets: Anthropic, Google, Microsoft, Amazon — yeh dream companies hain, ₹25L–₹50L deti hain India mein remote. Agar 5-6 projects aur strong DSA hai toh interview crack hota hai. Flipkart, Swiggy, Razorpay — yeh mid-tier hain, ₹15L–₹25L deti hain, competition thoda kam hai. TCS Digital, Cognizant GenC — safe option, ₹8L–₹12L, almost guaranteed placement. 2 saal experience ke baad US remote companies target karna. LinkedIn pe apply karo, remote.co pe dekho. US companies India se remote hire karti hain — salary $100k–$200k hoti hai yaani ₹80L–₹1.6Cr. Indore mein baith ke yeh earn karna possible hai.Salary Timeline MCA ke baad pehli job mein ₹12L–₹25L milega depending on company. 2 saal experience ke baad ₹25L–₹40L. 5 saal ke baad ₹40L–₹80L. 8–10 saal ke baad ₹1Cr+ possible hai — ya toh India mein senior role se, ya US remote se, ya apna startup se. Daily Routine (Abhi Se) Subah college jao, dhyan se padho, notes khud likho. Afternoon mein assignments karo — copy mat karna kyunki samajh mein nahi aayega. Shaam ko 4–6 baj ke beech side skill seekho — YouTube ya practice. Raat ko 1 LeetCode problem solve karo, chahe easy ho. Week mein ek baar GitHub pe kuch push karo. Sabse Important Rules College assignments khud karo — copy karne se marks toh milte hain par skills nahi. Projects genuinely banao — recruiters samjh jaate hain copy-paste kab hua hai. Attendance 90%+ rakhna — internal marks mein counted hota hai. CGPA 9+ maintain karna — yeh GATE aur top companies ke liye threshold hai. Aur sabse important — daily code likhna, chahe 30 minute ho. Consistency hi tera sabse bada weapon hai.
